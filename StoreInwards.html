STORE INWARDS¶

The Store Inwards screen on the platform represents the products to be delivered to the store inventory in the given time range.

Table¶

  • The default time range for which insights are presented on first load is the current week. This can be changed with the help of the time filter in the top right portion of the screen.
  • The user can also select the category of products for which the insights are needed.
  • There is also an option of searching a specific product in order to get insights for the same.
  • Product Name - Gives the name of the product that is expected in the store inventory.
  • Item Code - The SKU code of the product for the given store.
  • Item Count - Gives the number of units of the product selected that is to be received by the store.
  • Expected Date - Gives the tentative date for the shipment of the product to be delivered to the store.
