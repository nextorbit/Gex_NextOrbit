TOP SELLERS
============


The ‘Top Sellers’ screen of the platform provides details of the top products based on the Revenue generated or the total number of transactions made by each product.The default time range for which insights are provided on the first load is the current week. This can be changed using the time filter in the top right portion of the screen.

-------
